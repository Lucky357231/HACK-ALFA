КЭП на токене

Ключи электронной подписи хранятся на специальном USB-токене, этот токен вставляется в компьютер. Для формирования электронной подписи достаточно ввести ПИН-код от токена.

Преимущества
  • Самый безопасный способ.
  • Можно подписывать абсолютно все документы, не только в банке, но и в других организациях.
  • Самый простой путь использования при работе на ПК
Недостатки
  • Неприменим на мобильном телефоне.
  • Требует настройки ПО на ПК.
  • Требует наличия токена — специального устройства для работы с ключами электронной подписи.
  • Для директора получение бесплатно, для физ.лица (не директора фирмы) — на коммерческой основе.
  • Получение только офлайн в отдельных организациях — аккредитованных удостоверяющих центрах.
  • Для использования физлицом (не директором фирмы), требуется дополнительно использовать машиночитаемую доверенность (выпускать, размещать, загружать).

КЭП в приложении

Ключи электронной подписи хранятся в отдельном мобильном приложении \ на специальном USB-токене с интерфейсом NFC. Для формирования электронной подписи достаточно ввести ПИН-код \ ввести ПИН-код и приложить токен с NFC к мобильному устройству.

Преимущества
  • Самый безопасный способ.
  • Можно подписывать абсолютно все документы в банке.
  • Подходит как для работы как на мобильном устройстве, так и на ПК.
Недостатки
  • Получение только офлайн в отдельных организациях – аккредитованных удостоверяющих центрах на коммерческой основе.
  • Для подписи директора требуется USB-токен с интерфейсом NFC.
  • Для использования физлицом (не директором фирмы), требуется дополнительно использовать машиночитаемую доверенность (выпускать, размещать, загружать).
  • Можно подписывать только документы от банка.

PayControl

Ключи электронной подписи хранятся в банковском мобильном приложении. Для подписания клиент вводит свой секретный код или использует биометрический датчик смартфона.

Преимущества
  • Простота использования.
  • Независимость от сотовой сети.
  • Надежная защита от мошенников.
  • Онлайн-подключение.
Недостатки
  • Для использования в интернет-банке нужен мобильный телефон с приложением.
  • Можно подписывать базовые документы(платежи, депозиты и т.д, 99% всех документов), для документов особой важности (по кредитным сделкам, для деривативов, налоговой отчетности и т.д.) данного вида подписи недостаточно.
